Hockeypuck

An OpenPGP keyserver developed in Go

26 April 2014

OpenPGP Keyservers

  • Directory service for delivering updates to public key material.
  • Not a trusted system.
  • Clients (GnuPG, etc.) must validate key material.

Typical use

  • Importing key material for a new contact
  • Discovering new updates (signatures, revocations, etc.) for known contacts.

Quick Demo

  • GnuPG interface
  • Web interface
